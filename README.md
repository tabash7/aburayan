# Alretaj — Enterprise Solutions Website

A professional, fully responsive corporate solutions website built with pure HTML, CSS, and JavaScript.

## Features

- 🎨 Dark-themed premium design with gradient accents
- ✨ Animated particle network hero background
- 📊 Animated counters and scroll-triggered animations
- 🏭 Industry-specific tabbed solutions (Finance, Healthcare, Retail, Manufacturing, Logistics)
- 💬 Testimonials carousel with auto-play
- 📬 Contact form with validation
- 🔢 Statistics section with count-up animations
- 📱 Fully responsive (mobile, tablet, desktop)
- ♿ Accessible (ARIA labels, semantic HTML)
- ⚡ No external dependencies — 100% compatible with GitHub Pages

## Structure

```
├── index.html        # Main page
├── css/
│   └── style.css     # All styles
└── js/
    └── main.js       # All interactions
```

## Deployment

This site is designed for GitHub Pages deployment. Simply push to the `main` branch and enable GitHub Pages in repository settings.
